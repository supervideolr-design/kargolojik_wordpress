== Kurulum ==

1. WordPress admin paneline giriş yapın
2. Görünüm > Temalar > Yeni Ekle'ye gidin
3. 'Tema Yükle' butonuna tıklayın
4. kargolojik-theme.zip dosyasını seçin
5. 'Hemen Kur' ve 'Etkinleştir' butonlarına tıklayın

== CSV İçe Aktarma ==

WordPress Admin > Kargolojik > CSV İçe Aktar bölümünden şube verilerinizi yükleyebilirsiniz.

CSV Formatı:
Sirket_Adi;Sube_Adi;Sehir;Ilce;Adres;Telefon_1;Telefon_2

== Kısa Kodlar ==

[kargolojik_search] - Arama kutusunu gösterir
[kargolojik_branches] - Şube listesini gösterir

== Özellikler ==

- Şube arama ve listeleme
- Kargo şirketi filtreleme
- Şehir ve ilçe bazlı filtreleme
- Yardım konuları
- Responsive mobil tasarım
- SEO uyumlu
- Türkçe karakter desteği
